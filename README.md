# B&W Pixelator
A simple web app that transforms images into black and white pixelated versions.
## Setup
1. Install dependencies
```bash
npm install
```
2. Start development server
```bash
npm run dev
```
## Usage
1. Upload an image
2. Use the slider to adjust pixel size
3. Download the pixelated result
## Tech Stack
- React + TypeScript
- Vite
- Tailwind CSS
